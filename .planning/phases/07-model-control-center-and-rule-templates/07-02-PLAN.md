# Plan 07-02

## Objective

在现有 Web App 壳上实现模型中心和规则模板 UI，并把它们接入工作台任务入口。

## Tasks

1. 将前端从单页面壳扩成“workspace + settings/control center”结构
2. 实现模型中心页，至少支持：
   - 查看 provider / model
   - 调整 `high` / `medium` / `low` / `disabled`
   - 启用 / 禁用模型
3. 实现规则模板页，至少支持：
   - 创建模板
   - 选择场景或 preset
   - 配置默认协调模型
   - 为不同 role / responsibility 指定模型
   - 编辑备注说明
4. 在主工作区增加模板选择或模板命中结果展示入口
5. 把模板、协调模型和分配结果显示在任务结果或 metadata 面板里

## Key Files (expected)

- `frontend/src/App.tsx`
- `frontend/src/components/settings/*`
- `frontend/src/components/model-control/*`
- `frontend/src/components/rule-templates/*`
- `frontend/src/lib/api.ts`
- `frontend/src/types.ts`
- `frontend/src/styles.css`

## Verification Focus

- 用户可以在前端调整模型优先级并看到持久化结果
- 用户可以创建规则模板并把不同职责映射到不同模型
- 主工作区能显示本次任务使用的模板、协调模型和角色分配结果
