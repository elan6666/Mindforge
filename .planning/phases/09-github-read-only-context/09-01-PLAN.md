# Plan 09-01

## Objective

实现 GitHub 只读上下文读取、摘要化 schema 与任务执行链集成。

## Tasks

1. 设计 GitHub repository / issue / pull request 的摘要 schema
2. 增加 GitHub read-only service，优先支持公开仓库或本地可用认证
3. 扩展 `TaskRequest` 支持 GitHub 关联字段
4. 在 `TaskService` 中读取 GitHub 上下文并写入 task metadata
5. 增加只读 GitHub 查询 API，供前端独立拉取或预览

## Key Files (expected)

- `app/backend/schemas/github_context.py`
- `app/backend/services/github_context_service.py`
- `app/backend/api/routes/github.py`
- `app/backend/schemas/task.py`
- `app/backend/services/task_service.py`

## Verification Focus

- 任务能接受 GitHub 引用参数
- GitHub 上下文能被写入 task metadata
- 只读接口能返回摘要化对象而不是原始大响应
