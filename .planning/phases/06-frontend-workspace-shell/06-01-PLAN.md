# Plan 06-01

## Objective

搭建前端 app shell、基础路由和左侧导航，使 `Mindforge` 拥有类似 Codex/OpenHands 的工作台骨架。

## Tasks

1. 初始化前端工程骨架和运行方式
2. 建立全局 layout、sidebar、top-level workspace route
3. 实现导航入口：new task、history、projects、presets、settings
4. 为会话/任务历史提供基础列表区和状态占位
5. 让整体布局在桌面和移动端都能正常工作

## Key Files (expected)

- `frontend/package.json`
- `frontend/src/app/*`
- `frontend/src/layout/*`
- `frontend/src/components/sidebar/*`
- `frontend/src/pages/*`

## Verification Focus

- 前端能够独立启动
- 能渲染完整工作台壳
- 侧边栏与主工作区布局稳定
