# Plan 08-01

## Objective

实现审批触发、审批记录和阻塞式审批交互链。

## Tasks

1. 设计审批相关 schema 与 SQLite 持久化结构
2. 增加高风险动作判定逻辑和审批触发器
3. 在任务执行链中支持 `pending_approval` 状态和审批等待态响应
4. 增加审批 API，至少支持：
   - 查询待审批项
   - 批准
   - 拒绝
5. 把审批结果回写到 task metadata 和 approval history

## Key Files (expected)

- `app/backend/schemas/approval.py`
- `app/backend/storage/*`
- `app/backend/services/approval_service.py`
- `app/backend/api/routes/approvals.py`
- `app/backend/services/task_service.py`

## Verification Focus

- 高风险任务会进入等待审批状态
- 批准和拒绝都有结构化记录
- 审批通过后任务能继续或正确结束
