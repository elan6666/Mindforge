# Plan 10-01: Paper Role Contracts and Academic Context

## Objective

Define the paper revision stages, role responsibilities, academic-context IO, and request metadata needed for journal-aware paper revision.

## Tasks

- Add academic request fields to `TaskRequest`: journal name, journal guideline URL, and reference paper URLs.
- Add schemas for journal guideline summaries, reference paper summaries, and aggregate academic context.
- Implement a lightweight academic context service that fetches cleaned page titles and excerpts.
- Inject academic context into task metadata and orchestration prompts.
- Define paper-specific role contracts for standards analysis, revision, style review, content review, and final review.

## Verification

- Unit tests cover academic context metadata persistence.
- Orchestration tests verify stage prompts receive academic context.
- Existing task submission behavior remains unchanged for non-paper presets.
